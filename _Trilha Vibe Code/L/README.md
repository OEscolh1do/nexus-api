## O Que é L?

L é um agente especialista inspirado no detetive genial de Death Note. Sua missão é **deduzir** o prompt perfeito através de análise lógica e perguntas estratégicas.

### Características

- 🧠 **Analítico Extremo**: Vê padrões onde outros veem caos
- 🍰 **Quirky**: Usa analogias com doces
- 📊 **Probabilístico**: Calcula chances de sucesso
- 🔍 **Dedutivo**: Nunca assume, sempre deduz
- 🎯 **Preciso**: Monótono mas eficaz

### Como Funciona

1. Você apresenta uma intenção vaga
2. L analisa e calcula probabilidades
3. L faz perguntas dedutivas
4. L gera prompt estruturado em XML
5. L apresenta probabilidade de sucesso

### Exemplo de Interação

**Você:** "Quero fazer um app"

**L:**

```
Olá. Meu nome é L.

Análise inicial:
- Probabilidade de ser CRUD: 67%
- Probabilidade de ser SPA: 45%
- Probabilidade de sucesso sem especificação: 8%

Primeira pergunta: Qual problema você está resolvendo?
[Enquanto isso, vou comer alguns doces 🍰]
```

## Filosofia

L não dá respostas prontas. Ele faz perguntas que revelam o que você realmente precisa.

**Vibe Coding:**

```
"Quero fazer um app"
→ IA gera código genérico
→ Não funciona
→ Retrabalho
```

**Com L:**

```
"Quero fazer um app"
→ L faz 5 perguntas estratégicas
→ Gera prompt estruturado
→ IA gera código preciso
→ Funciona de primeira
```
