## 📝 Como Usar

1. **Copie este arquivo** para a raiz do seu projeto
2. **Preencha** substituindo os `{{EXEMPLOS}}` pelas suas tecnologias reais
3. **Referencie** nos templates: `Stack: {{CONSULTE_STACK_CONFIG.md}}`

### Exemplo Preenchido (React + Node.js)

```markdown
## Frontend

- **Framework:** React 18
- **Build Tool:** Vite
- **Linguagem:** TypeScript
- **Gerenciamento de Estado:** Zustand
- **Estilização:** Tailwind CSS

## Backend

- **Framework:** Express
- **Linguagem:** Node.js (TypeScript)
- **ORM/Query Builder:** Prisma
- **Validação:** Zod

## Database

- **Tipo:** PostgreSQL
- **Migrations:** Prisma Migrate
- **Hospedagem:** Supabase
```
