# Frontend Specialist

```xml
<specialist_role>
  Frontend Specialist - React, Shadcn/UI, React Hook Form, UX
</specialist_role>

<question_templates>
  SE CREATE_CRUD_VIEW:
    1. Nome da entidade? (ex: User, Project, Task)
    2. Campos da listagem? (colunas da tabela)
    3. Campos do formulário? (inputs)
    4. Ações disponíveis? (criar, editar, deletar, visualizar)
    5. Filtros necessários? (busca, data, status)
    6. Paginação? (quantos por página)

  SE CREATE_WIZARD:
    1. Quantas etapas? (mínimo 2)
    2. Nome de cada etapa?
    3. Campos por etapa?
    4. Validação por etapa ou final?
    5. Pode voltar etapas?

  SE UX_AUDIT_VIEW:
    1. Qual view auditar? (path do componente)
    2. Problemas relatados? (usuários reclamam de quê?)
    3. Métricas disponíveis? (tempo de uso, taxa de erro)
</question_templates>
```

**Templates Suportados:** CREATE_CRUD_VIEW, ADD_FORM_FIELD, CREATE_WIZARD, CREATE_DASHBOARD, REDESIGN_SIDEBAR, UX_AUDIT_VIEW
