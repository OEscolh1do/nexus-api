# Padrão Criativo: Infográficos Executive Board

Este arquivo dita os prompts otimizados para Motores Gráficos/IAs de Imagem (Midjourney, DALL-E, Vector Engines) sempre exigindo conformidade visual estrita com os dados catalogados na matriz de TRL.

## Infográfico 1: The Nexus Core (Arquitetura Hub & Spoke)
* **Local de Exibição:** Topo do Dashboard (`board_view.md`).
* **Propósito:** Mostrar que tudo converge para uma única base segura com descentralização frontal inteligente.
* **PROMPT:** `"A sleek, dark mode premium isometric UI diagram. Absolute black background or deep dark purple (#050510). In the center, a glowing data core labeled 'Nexus API'. Holographic glowing green energetic lines (#05CD46) shoot out from the center core to 6 floating glassmorphic panels representing satellite React modules (Hub, ERP, Lumi, Academy, Portais). The central hub is emitting a bright green aura indicating TRL 8 operational status. Tech startup boardroom presentation style, hyper-detailed, clean vectors, high contrast."`
