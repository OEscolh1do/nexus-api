# Fluxo de Trabalho: Sync de Documentação (.agent/context.md)

Toda vez que uma nova quebra de paradigma (nova tecnologia inserida, novo padrão global do Hub, ou novo redirecionamento do roteador massivo) for realizada:

1. Modifique ou atualize o `.agent/context.md` de cima pra baixo atestando o que foi incluído.
2. Não perca os tópicos essenciais explicativos ali marcados.
3. Certifique-se de que agentes subsequentes lerão e adotarão o novo padrão arquitetural sem perder eficiência.
