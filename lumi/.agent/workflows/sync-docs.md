# Fluxo de Trabalho: Sincronia de Documentação Core (Sync Docs)

Sempre que a arquitetura do Lumi mudar significativamente (ex: Nova Aba adicionada, novo Slice criado no Zustand, Zod alterado de forma drástica, ou Nova Rota no Backend consumida), o `.agent/context.md` DEVE ser atualizado.

## Etapa Única: Atualização Sistêmica
1. Leia o `.agent/context.md` e localize onde a nova tecnologia/biblioteca ou módulo se encaixa.
2. Adicione ao `Glossário de Domínio` se envolver uma nova Sigla Técnica (ex: "Payback Simples", "TIR", "Fator Sombra").
3. Salve e certifique-se de que a IA compreende a alteração na sua janela de memória de longo prazo.
